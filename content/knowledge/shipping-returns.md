# Shipping, Delivery, and Returns

Coremont ships across Europe with tracked delivery and clear timeframes.

Delivery timelines:
- Standard delivery: 5–7 working days
- Scheduled delivery: 4–6 working days
- White glove setup: 7–10 working days (includes placement and assembly)

Returns:
- Returns are accepted within 30 days for unused equipment
- Return labels and pickup can be arranged via support
- Bundles must be returned as a complete set

Need support? Contact support@coremont.eu for delivery updates or pickup scheduling.
